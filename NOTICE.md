# Third-party notices

This project builds on Selkies Project's `nvidia-egl-desktop` image and applies a small iPadOS pointer-input compatibility modification. Selkies components retain their MPL-2.0 notices.

The desktop launcher downloads a pinned revision of `cryinkfly/Autodesk-Fusion-360-on-Linux` from Codeberg. That project and downloaded installer retain their own license and copyright notices.

No Autodesk software is included in this repository or container image. Autodesk Fusion is installed separately by the end user.
